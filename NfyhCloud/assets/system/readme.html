《南方院后云服务平台服务条款》

保密约定：南方院后云服务平台保证用户信息及会员信息仅提供平台医生及所属群管理员（医生助理）可看，其他任何人均不能看到也无权使用。 我申请成为南方院后云服务平台会员，在南方院后云服务平台上接受我的主诊医生对我院外情况的监护、指导服务。我同意南方院后云服务平台如下约定：

1、我接受并遵守南方院后云服务平台服务条款。

2、我愿意根据自己的需求选择服务的类型。

3、我确认我在南方院后云服务报到中提交的信息以及今后提供的信息均是真实的情况。我同意参加并接受主诊医生在南方院后云服务平台上的院后疾病管理服务。我愿意按照医生指导，真实的记录平时的身体状况，服药情况，生活状况。

4、我同意医疗是严肃的科学，尽管我是医生的老病人，医生对我的病情有相当的了解，但医生在网上给我的回复都是建议，我们按专业建议看待，不擅自采用，不引用这些医生对我的关怀类的信息服务做为质疑医生给我的处置方案的证据。我保证会按医疗程序看病。

5、我知道这个南方院后云服务，不包括紧急情况下给予干预意见，例如：病情发生较突然的变化时，感觉症状加重时，在这种紧急情况发生后均需要立即去当地有能力的医院就诊，而不可在网上等待建议。正确的做法是马上求助当地医院及拨打120电话。

6、我知道并同意我在网上登记的部分个人信息及疾病信息被我的主诊医生及南方院后云服务的所属群管理员看到，并允许他们按照本服务条款的约定使用。

7、我知道这个南方院后云服务服务医生需要了解我在院外的医嘱执行情况，日常生活中饮食、睡眠分析、运动情况、症状体征、情绪、心理等生活细节，我愿意定期按要求提交或当医生或医生助理问起这些情况时，如实配合。

8、我同意并支持医患之间应以相互理解、信任为基础，共同对抗疾病。

南方院后云服务平台向您提供的服务承诺：

1、本服务是由您的主诊医生及医生助理一起给您提供的医疗信息类指导服务。

2、南方院后云服务给您提供稳定的记录上传病情的专用平台，该平台由您及您授权人、您的主诊医生及医生助理共同使用。

3、南方院后云服务及您的主诊医生共同保证您信息的安全，除本服务条款中约定的人员外，不得透漏给其他任何人。

4、在已经隐去所有您个人识别信息后的网站页面，您也有权要求设置为全隐私状态，不与其他病友分享疾病及治疗、康复情况。

5、南方院后云服务保证您上传的资料，主诊医生能得到通知并随时、顺利的看到。

6、主诊医生会查看您上传或医生助理与您沟通后帮您上传的恢复情况，给您相应的指导建议。

7、根据您疾病的情况、康复阶段，医生及医生助理一起工作保证您了解并理解相关的治疗方式及可能会出现的情况、如何预防、如何发现等知识。

8、医生助理会按主诊医生给您安排的评估调查、复查项目、复诊安排等提前通知您，并约定具体操作时间、注意事项等。

9、如果您因为确切原因不满意南方院后云服务及主诊医生的服务或因特殊情况（痊愈、突发病情加重）无法、无必要再使用院后疾病管理服务，可与南方院后云服务平台协商。

本服务条款最终解释权归南方院后云服务平台所有。
